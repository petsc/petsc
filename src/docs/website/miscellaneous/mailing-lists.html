    Please continue to use petsc-maint@mcs.anl.gov for maintainance/bug-report/[installation issues requiring sending in configure.log]. However we have the following mailing lists with public archives for general use.

    • petsc-announce@mcs.anl.gov : archives : For announcements regarding PETSc releases.
    • petsc-users@mcs.anl.gov         : archives : For PETSc users.
    • petsc-dev@mcs.anl.gov            : archives : For petsc-dev users.

    To subscribe to any of the above lists, send e-mail to majordomo@mcs.anl.gov with the message "subscribe list-name" For eg: - to subscrbe to all the above lists, send the following e-mail

              To: majordomo@mcs.anl.gov
              Subject: [not required]
                  subscribe petsc-users
                  subscribe petsc-announce
                  subscribe petsc-dev

    Notes:

    • petsc-announce is an announcement only list. [i.e users cannot post on this list]
    • petsc-users, petsc-dev are subscriber-only lists. [i.e to post to this list - you have to subscribe first]
    • To unsubscribe send email to 'majordomo@mcs.anl.gov' with the message "unsubscribe list-name"
    • the lists have limits on message sizes - so if you have to send in attachments - most likely the appropriate place is petsc-maint@mcs.anl.gov - not the mailing list.
