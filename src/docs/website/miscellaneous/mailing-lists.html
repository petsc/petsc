Please continue to use petsc-maint@mcs.anl.gov for maintainance/bug-report/[installation issues requiring sending in configure.log]. However we have the following mailing lists with public archives for general use.

• petsc-announce@mcs.anl.gov : subscribe/unsubscribe : archives : For announcements regarding PETSc releases.
• petsc-users@mcs.anl.gov : subscribe/unsubscribe : archives : For PETSc users.
• petsc-dev@mcs.anl.gov : subscribe/unsubscribe : archives : For petsc-dev users. This mostly PETSc developers but also followed by others who are interested in the development process of PETSc.

Do not send configure and installation questions to the petsc-users mailing list. Always send them (and include configure.log and make.log) to petsc-maint@mcs.anl.gov

Notes:

  • petsc-announce is an announcement only list. [i.e users cannot post on this list]
  • petsc-users and petsc-dev are subscriber-only lists. [i.e to post to this list - you have to subscribe first]
  • the lists have limits on message sizes - so if you have to send in attachments - most likely the appropriate place is petsc-maint@mcs.anl.gov - not the mailing list.
  • The mailing lists are migrated from majordomo to mailman in Jan-09. So links to archives, and subscription procedures have changed since then - as indicated above.
