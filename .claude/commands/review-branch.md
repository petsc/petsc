Determine the target-branch between release or main. Review a local branch's changes against it. Adhere to @CLAUDE.md while reviewing. First run `git fetch origin target-branch` to ensure the base is up to date. If a branch name is given in $ARGUMENTS, use `git diff origin/target-branch...$ARGUMENTS`. Otherwise, use `git diff origin/target-branch...HEAD` for the current branch. Follow steps 3–4 from @.claude/commands/review-mr.md.
