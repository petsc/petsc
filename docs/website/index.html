The current version of PETSc is 2.1.5; released  January 27, 2003. 
Web pages last updated February 24, 2003

PETSc is a suite of data structures and routines for the scalable (parallel) solution of scientific applications modeled by partial differential equations.  It employs the MPI standard for all message-passing communication.  

  • Scientific applications that use PETSc
  • Features of the PETSc libraries
  • Related packages that use PETSc
    • Toolkit for Advanced Optimization (TAO).
    • ParPre - Additional Parallel Preconditioners
  • Packages that PETSc can optionally uses 
  • PETSc is part of the DOE SciDAC TOPS project in algorithms, software, and applications for the scalable solution of PDES.
